# Changelog

## 0.4.0 - New site map
- The site page now shows the site area by default. Bodies and allocations can be enabled as needed.
- Bodies are displayed on maps with dotted lines instead of filled areas
- Improvement Habitats now display data used for HU calculation in their subtable (Created habitats only; Enhancement habitat HU calculation coming soon)
- Fixed some incorrect HU calculations
- Fixed some incorrect habitat assignment in Sankey charts

## 0.3.0 - Habitat Improvement Chart
- Added Habitat Improvement Sankey Chart to site details page
- Re-arranged the Site Summary infromation to use space better
- Individual Trees are displayed seperately from other Area habitats

## 0.2.0 - Visual Update
- Charts are now accessable via tabs instead of modal buttons
- Habitat Tables have also been moved to tabs instead of being all on the same page
- Formatting changes to reduce whitespace so more data can be viewed at once
- Added Dark mode
- Switched to use LSOAs from 2021 and IMD data from 2025
- Added IMD Score Transfers Histogram to BGS Allocations page

## 0.1.0 - First Beta Release
